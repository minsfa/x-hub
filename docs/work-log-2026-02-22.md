## 초보자용: 개발 서버 실행 방법

**매번 개발을 시작할 때 터미널에서 아래 명령어를 실행해야 합니다.**

```bash
cd /Users/ykmin/Documents/code/x-hub/x-hub
npm run dev
```

- `cd ...` : 프로젝트 폴더로 이동
- `npm run dev` : Vite(프론트) + Express API(백엔드) 동시 실행

실행 후 브라우저에서 **http://localhost:3000** 접속.

### 자주 겪는 문제

| 증상 | 해결 방법 |
|------|----------|
| `EADDRINUSE: port 3001` | 이전 프로세스가 남아 있음. `lsof -ti:3001 \| xargs kill -9` 실행 후 다시 `npm run dev` |
| `localhost` 접속 안 됨 | `http://127.0.0.1:3000` 으로 시도 |
| DB 연결 오류 | Docker 실행 여부 확인: `docker compose up -d` |
